# 基于自编码器的手写数字生成

要求使用自编码器生成手写数字，并对隐空间中的向量进行可视化。 
在此分别使用了	基础Auto-encoder、CAE（Convolutional Auto-encoder）卷积自编码器、VCAE（Conditional Variational Auto-Encoder）条件变分自编码器。
